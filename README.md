## ğŸš€ Quick Start

### 1. Clone the Repository
```bash
git clone https://github.com/PradyumnaPrasad/NeuroMentor_AdaptiveLearningPlatform.git
cd NeuroMentor_AdaptiveLearningPlatform
```

### 2. Backend Setup

#### Create Virtual Environment
```bash
cd adaptive-learning-backend
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
```

#### Install Dependencies
```bash
pip install -r requirements.txt
```

#### Environment Configuration
```bash
# Copy and configure environment variables
cp .env.example .env
```

Edit `.env` with your settings:
```env
MONGODB_URL=mongodb://localhost:27017
MONGODB_DB=adaptive_learning
GEMINI_API_KEY=your_gemini_api_key_here
SECRET_KEY=your_secure_jwt_secret_key
ALGORITHM=HS256
ACCESS_TOKEN_EXPIRE_MINUTES=30
```

#### Start MongoDB
```bash
# Using Homebrew on macOS
brew services start mongodb-community

# Or run manually
mongod --dbpath /usr/local/var/mongodb
```

#### Run Backend Server
```bash
python -m uvicorn src.main:app --reload --host 0.0.0.0 --port 8000
```

### 3. Frontend Setup

#### Install Dependencies
```bash
cd ../client
npm install
```

#### Start Development Server
```bash
npm run dev
```

### 4. Access the Application

- **Frontend**: http://localhost:5173
- **Backend API**: http://localhost:8000
- **API Documentation**: http://localhost:8000/docs

## ğŸ”§ Configuration

### AI Model Settings
- **Gemini Model**: `gemini-2.0-flash` for fast, accurate responses
- **Question Batch Size**: 3 questions per practice session
- **Explanation Context**: Includes concept tags and student performance

### Reinforcement Learning
- **Q-Network Architecture**: Custom neural network for action selection
- **Reward System**: Points for correct answers, penalties for repeated mistakes
- **Experience Replay**: Buffer of 10,000 experiences for stable learning

- **Password Policy**: Minimum 50 characters (truncated for security)

- **CORS**: Configured for local development
